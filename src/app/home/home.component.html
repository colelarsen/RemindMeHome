The front end of this website was made using Angular with Bootstrap. The page you are currently seeing has no backend but you can check out the Reminders page if you want to see microservices in action. This page is also mobile friendly!

Reminders
The "Reminders" page was made for a chat bot that I developed. This page allows users of a group chat to submit and edit reminders. The microservices that handle these requests are hosted on the same EC2 server that this webserver is running on. The microservices handle the requests and interact with a DynamoDB NoSQL database. The users are reminded by a chat bot created with discord.js. The chat bot interacts with the same microservices to request reminders and when the time expires it sends the user a message! There are two microservices running to control the functionality of this page and the user accounts. One microservice handles user login / account creation / authorization. The other microservice handles get/put/post requests dealing with reminders. They work in tandem to ensure only authenticated users can change the reminders they own.

Yugioh Scraper

The "Yugioh Scraper" page was made so that you can check out a personal project I made in early 2019. I converted the project into a microservice so that it is interactable without having to download the project from github. This project solves the issue of trying to find pdfs of a card by allowing users to enter card names and download them straight to their computer. The microservice was made using Express and it uses web scraping to find links to pdfs!
