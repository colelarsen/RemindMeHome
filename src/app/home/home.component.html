About This Website

This website and all custom functionality was developed entirely by me. The webserver hosting this website is a nginx server running on an AWS EC2 server. The front end of this website was made using Angular with Bootstrap. The page you are currently seeing has no backend but you can check out the Reminders page if you want to see a microservice in action. This page is also mobile friendly!

Pages

Reminders
The "Reminders" page was made for a microservice that I developed. This page allows users of a group chat to submit and edit reminders. The microservice that handles these requests is hosted on the same EC2 server that this webserver is running on. The microservice handles the requests and interacts with a Firebase NoSQL database. The users are reminded by a chat bot created with discord.js. The chat bot interacts with the same microservice to request reminders and when the time expires it sends the user a message and deletes the reminder!
